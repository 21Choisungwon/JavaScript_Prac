변수의 선언
    var를 통한 선언
        var 변수명;
        재선언 가능--> 문제점 , 재할당 가능

    let을 통한 선언
        let 변수명;

        재선언 불가능, 재할당 가능

    const을 통한 선언
        const 변수명;

        재선언 불가능, 재할당 불가능

변수 종류 정하기
    값을 나중에 다시 써야할 경우 --> 변수 사용
        해당 변수에 다른 값이 대입될 수 있어야 한다 --> let
        해당 변수에 최초로 대입된 값 외에는 다른값이 대입되면 안된다 -->const

const 는 읽기 전용 (read-only)으로 사용한다.
    변수 속에 저장된 값을 사용만 할 수 있다. --> 수정 불가능

                let my_name;

    4. 키워드(예약어) 사용 불가능
        let var; //불가능

    1. (소괄호 안의 값) 인자값의 갯수, 타입

    2. 함수의 동작

    3. 함수 사용 결과 (return / 반환) 값의 타입
        사용자가 prompt 창에 입력한 문자열 값(String타입)

``기호
    ``기호 안에 ${}를 사용하면 {}안쪽은 자바스크립트 명령어로 동작하는 곳이고,
    변수라면 변수속의 값, 함수라면 함수 사용 결과 값 을 ``안의 하나의 문자열로 만들어준다.

특수문자를 포함한 문자열 만들기
    ex) `최성원은 생각했습니다 `배고프다` 라고 ` 
    ex) "최성원은 말했습니다 "배고프다" 라고"
    ex) '나는
        오늘이 행복하다'

    문자열 내에서 특수문자를 표현하기 위해서는 제어문자를 활용한다

제어문자    escape
    \' : 문자열 값 ' 를 의미(문자열을 알려주는 역할에서 벗어난다)
    \" : 문자열 값 " 를 의미
    \n : 엔터를 의미
    \t : Tab 키를 의미 
    \\: \를 의미
